Project Overview

For this project, I utilized Node.js to extract the data from the Binance API, and then fed it into a Spark streaming application. The goal of the application was to calculate real-time metrics such as mean, variance, and standard deviation of a selection of cryptocurrencies. To visualize this data, I created a web server using Python's Flask library, and utilized the BOkeh library to plot the data. This approach allowed for the easy and dynamic visualization of crypto market trends, making it simpler to make informed decisions based on current data. While working on this project, I gained valuable experience in leveraging APIs to extract real-time data, using Node.js to develop backend systems, and utilizing Flask and BOkeh libraries for web development and data visualization. Additionally, I developed my skills in Spark streaming for real-time data processing, which is a valuable tool for analyzing large-scale data sets. Overall, this project provided a unique opportunity to apply various programming languages and tools to solve a real-world problem, and has helped me to become a more versatile and capable developer.

Java 3D pathfinder

Java 3D Pathfinder is a tool that demonstrates pathfinding algorithms in a 3D environment. With its intuitive interface and advanced capabilities, the tool offers a unique and engaging way to showcase different pathfinding techniques using the powerful Java programming language.

Project Overview

Java 3D Pathfinder is an innovative tool that allows users to explore and compare different pathfinding algorithms in an immersive 3D environment. This powerful tool, designed using the Java programming language, provides a unique and engaging way to showcase the DFS, Dijkstra, and A* algorithms, all of which are critical components of pathfinding. The tool animates the expanded nodes, using a JavaFx powered interface, and highlights the found path between two points. It uses a 3D representation to depict one floor at a time, with arrows in each cell indicating whether the up or down cells are obstacles or clear. The algorithms factor in this information to identify the path, even if it spans multiple floors. Overall, Java 3D Pathfinder is a tool for students, developers, and anyone interested in pathfinding algorithms and their practical applications.

Through this project, I gained valuable experience in both the Java programming language and the JavaFX library. I became proficient in utilizing various programming concepts such as data structures, algorithms, and object-oriented programming to design and develop an interactive and immersive tool. Moreover, I gained a deeper understanding of pathfinding algorithms such as DFS, Dijkstra, and A* and their practical applications. I learned how these algorithms work, how to implement them, and how to compare their performance. Additionally, I became familiar with the challenges and complexities involved in developing a tool that can display a 3D environment and animate pathfinding algorithms in real-time.
